# 8x8 方塊連線消除遊戲 - 規格說明書

## 1. 專案概述

- **專案名稱**：8x8 方塊連線消除遊戲
- **版本**：v1.0
- **類型**：網頁單機遊戲
- **核心功能**：結合《1010!》消除機制與俄羅斯方塊形狀的策略消除遊戲
- **目標用戶**：休閒遊戲玩家

## 2. UI/UX 規格


## 3. 功能規格

### 3.1 方塊系統

- 7 種 Tetromino 形狀 (I、J、L、O、S、T、Z)
- 每塊佔 4 格
- 4 種旋轉角度 (順時針 90°)
- 旋轉中心：I 形第 2 格，其他依標準

### 3.2 生成規則

- 每次生成 3 個不同形狀
- Fisher-Yates 洗牌後取前 3 種
- 5×5 預覽容器顯示

### 3.3 操作方式

- **滑鼠拖曳**：從備選區拖曳至棋盤
- **觸控拖曳**：支援 touch 事件
- **旋轉按鈕**：點擊旋轉對應方塊

### 3.4 放置邏輯

- 拖曳時高亮有效位置 (綠色半透明)
- 衝突或超出邊界：返回原位 + 紅色閃爍

### 3.5 消除規則

- 橫列或直欄填滿 8 格即消除
- 最多同時 16 條 (8 行 + 8 列)
- 消除後無重力掉落
- 單次掃描，無連鎖

### 3.6 遊戲結束判斷

每次放置後檢查備選區每個方塊：
- 測試 4 種旋轉角度
- 掃描所有可能置入位置
- 若全部無法放置 → Game Over

### 3.7 計分系統

| 動作 | 分數 |
|------|------|
| 放置方塊 | +4 分 |
| 消除 1 條線 | +30 分 |
| 消除 2 條線 | +75 分 |
| 消除 3 條線 | +150 分 |
| 消除 4 條線 | +300 分 |
| 消除 k≥5 條線 | 300 × 2^(k-4) |

### 3.8 儲存

- localStorage 鍵值：`bestScore`

## 4. 畫面規格

### 4.1 開始畫面

- 標題：「8x8 PUZZLE」
- 「開始遊戲」按鈕

### 4.2 遊戲進行中

- 頂部：分數、最高分
- 中部：8x8 棋盤

### 4.3 結束畫面

- 「GAME OVER」標題

- 最高分
- 「重新開始」按鈕
