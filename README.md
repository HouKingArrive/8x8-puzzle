# 8x8 Puzzle 俄羅斯方塊消除遊戲

結合《1010!》消除機制與俄羅斯方塊形狀的策略消除遊戲。

## 🎮 遊戲玩法

- 將方塊拖曳到 8×8 棋盤上
- 填滿水平或垂直 8 格即可消除得分
- 方塊無法放置時遊戲結束

## ⭐ 功能特色

- 7 種經典 Tetromino 方塊形狀
- 支援旋轉功能
- 消除連線得分動畫
- 自動儲存遊戲進度
- PWA 應用，可離線遊玩
- 響應式設計，支援手機與電腦
